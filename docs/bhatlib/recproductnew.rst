recproductnew
==============================================
Purpose
----------------
Performs a new variant of recursive product on a lower triangular matrix.

Format
----------------
.. function:: dS = recproductnew(dS)

    :param dS: Lower triangular matrix.
    :type dS: matrix

    :return dS: Updated matrix after recursive product.
    :rtype dS: matrix

Library
-------
bhatlib
