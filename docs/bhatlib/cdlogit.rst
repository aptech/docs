cdlogit
==============================================
Purpose
----------------
Computes the cumulative distribution function (CDF) of the logit distribution.

Format
----------------
.. function:: p = cdlogit(x)

    :param x: Evaluation points.
    :type x: scalar or vector

    :return p: Computed CDF values.
    :rtype p: scalar or vector

Library
-------
bhatlib
