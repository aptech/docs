gradlogsum
==============================================
Purpose
----------------
Computes the gradient of the log-sum of exponentials with respect to input values.

Format
----------------
.. function:: grad = gradlogsum(v)

    :param v: Vector of values.
    :type v: vector

    :return grad: Gradient values.
    :rtype grad: vector

Library
-------
bhatlib

------
