newcholparmconst
==============================================
Purpose
----------------
Computes a constant Cholesky parameterization.

Format
----------------
.. function:: S = newcholparmconst(sstar)

    :param sstar: Input parameter vector.
    :type sstar: vector

    :return S: Cholesky parameterization matrix.
    :rtype S: matrix

Library
-------
bhatlib
