ggradm
==============================================

Purpose
----------------

Computes gradients for the M matrix used in gradient calculations for covariance matrices.

Format
----------------
.. function:: g = ggradm(M)

    :param M: Input matrix M.
    :type M: matrix

    :return g: Gradient matrix.
    :rtype g: matrix

Library
-------
bhatlib

Source
------
matgradient.src
