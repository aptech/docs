gLA
==============================================
Purpose
----------------
Computes the gradient of a lower triangular matrix L with respect to matrix A.

Format
----------------
.. function:: g = gLA(L, A)

    :param L: Lower triangular matrix.
    :type L: matrix

    :param A: Input matrix.
    :type A: matrix

    :return g: Gradient matrix.
    :rtype g: matrix

Library
-------
bhatlib

Source
------
matgradient.src
