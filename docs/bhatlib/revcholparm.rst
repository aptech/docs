revcholparm
==============================================
Purpose
----------------
Computes the reverse Cholesky parameterization from a Cholesky matrix.

Format
----------------
.. function:: sstar = revcholparm(L)

    :param L: Cholesky factor matrix.
    :type L: matrix

    :return sstar: Parameter vector recovered from L.
    :rtype sstar: vector

Library
-------
bhatlib
