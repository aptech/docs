rectcombs
==============================================
Purpose
----------------
Computes rectangular combinations from two vectors.

Format
----------------
.. function:: { sign, combnew } = rectcombs(a, b)

    :param a: Vector of first set of values.
    :type a: vector

    :param b: Vector of second set of values.
    :type b: vector

    :return sign: Vector of signs associated with each combination.
    :rtype sign: vector

    :return combnew: Matrix of new combinations.
    :rtype combnew: matrix

Library
-------
bhatlib

Source
------
