rndskewn
==============================================
Purpose
----------------
Generates random variates from the skew-normal distribution.

Format
----------------
.. function:: r = rndskewn(n, alpha)

    :param n: Number of random variates to generate.
    :type n: scalar

    :param alpha: Skewness parameter.
    :type alpha: scalar

    :return r: Generated random variates.
    :rtype r: vector

-------
