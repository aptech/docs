cholspherparmconst
==============================================
Purpose
----------------
Computes the Cholesky parameterization using spherical coordinates with constant parameters.

Format
----------------
.. function:: S = cholspherparmconst(sstar)

    :param sstar: Parameter vector.
    :type sstar: vector

    :return S: Cholesky parameterization matrix.
    :rtype S: matrix

Library
-------
bhatlib

------
