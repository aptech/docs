gyjnonp
==============================================
Purpose
----------------
Computes the gradient of the nonparametric Yeo-Johnson transformation.

Format
----------------
.. function:: g = gyjnonp(lam, x)

    :param lam: Lambda parameter.
    :type lam: scalar or vector

    :param x: Input data vector.
    :type x: vector

    :return g: Gradient vector.
    :rtype g: vector

Library
-------
bhatlib

------
