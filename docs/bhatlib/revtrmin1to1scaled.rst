revtrmin1to1scaled
==============================================
Purpose
----------------
Applies the inverse of the scaled trmin1to1 transformation.

Format
----------------
.. function:: sstar = revtrmin1to1scaled(y, scal)

    :param y: Transformed data vector.
    :type y: vector

    :param scal: Scale parameter.
    :type scal: scalar

    :return sstar: Inverse transformed vector.
    :rtype sstar: vector

Library
-------
bhatlib
