gasymtosym
==============================================

Purpose
----------------

Converts an asymmetric gradient matrix to its symmetric equivalent for covariance derivatives.

Format
----------------
.. function:: gsym = gasymtosym(x)

    :param x: Asymmetric gradient matrix.
    :type x: matrix

    :return gsym: Symmetric gradient matrix.
    :rtype gsym: matrix

Library
-------
bhatlib
