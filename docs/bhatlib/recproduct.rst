recproduct
==============================================
Purpose
----------------
Performs recursive product on a lower triangular matrix.

Format
----------------
.. function:: dS1 = recproduct(dS)

    :param dS: Lower triangular matrix.
    :type dS: matrix

    :return dS1: Matrix after recursive product.
    :rtype dS1: matrix

Library
-------
bhatlib
