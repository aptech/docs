revtrmin1to1
==============================================
Purpose
----------------
Applies the inverse of the trmin1to1 transformation.

Format
----------------
.. function:: sstar = revtrmin1to1(y)

    :param y: Transformed data vector.
    :type y: vector

    :return sstar: Inverse transformed vector.
    :rtype sstar: vector

Library
-------
bhatlib
