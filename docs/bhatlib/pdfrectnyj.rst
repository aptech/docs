pdfrectnyj
==============================================
Purpose
----------------
Computes the PDF of a rectangular distribution with user-specified truncation.

Format
----------------
.. function:: p = pdfrectnyj(x, a, b)

    :param x: Evaluation points.
    :type x: scalar or vector

    :param a: Lower truncation point.
    :type a: scalar

    :param b: Upper truncation point.
    :type b: scalar

    :return p: PDF values.
    :rtype p: scalar or vector

Library
-------
bhatlib
