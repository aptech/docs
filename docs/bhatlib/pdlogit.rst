pdlogit
==============================================
Purpose
----------------
Computes the probability density function (PDF) of the logit distribution.

Format
----------------
.. function:: p = pdlogit(x)

    :param x: Evaluation points.
    :type x: scalar or vector

    :return p: PDF values.
    :rtype p: scalar or vector

Library
-------
bhatlib
