cholparm
==============================================
Purpose
----------------
Computes the Cholesky parameterization matrix.

Format
----------------
.. function:: S = cholparm(sstar)

    :param sstar: Input parameter vector.
    :type sstar: vector

    :return S: Cholesky parameterization matrix.
    :rtype S: matrix

Library
-------
bhatlib
