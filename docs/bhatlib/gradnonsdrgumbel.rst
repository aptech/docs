gradnonsdrgumbel
==============================================
Purpose
----------------
Computes the gradient of the complement of the standard deviation for the reversed Gumbel distribution.

Format
----------------
.. function:: grad = gradnonsdrgumbel(mu, beta)

    :param mu: Location parameter.
    :type mu: scalar

    :param beta: Scale parameter.
    :type beta: scalar

    :return grad: Gradient values.
    :rtype grad: vector

Library
-------
bhatlib
