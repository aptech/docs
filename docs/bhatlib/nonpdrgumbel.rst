nonpdrgumbel
==============================================
Purpose
----------------
Computes the non-probability density (complement) for the reversed Gumbel distribution.

Format
----------------
.. function:: npdf = nonpdrgumbel(x, mu, beta)

    :param x: Evaluation points.
    :type x: scalar or vector

    :param mu: Location parameter.
    :type mu: scalar

    :param beta: Scale parameter.
    :type beta: scalar

    :return npdf: Computed non-PDF values.
    :rtype npdf: scalar or vector

Library
-------
bhatlib
