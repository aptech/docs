notthere
==============================================
Purpose
----------------
Returns elements in `x` that are not present in `y`.

Format
----------------
.. function:: z = notthere(x, y)

    :param x: Vector of elements to check.
    :type x: vector

    :param y: Vector of elements to exclude.
    :type y: vector

    :return z: Vector of elements in `x` not present in `y`.
    :rtype z: vector

Library
-------
bhatlib
