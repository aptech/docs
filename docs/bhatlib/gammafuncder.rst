gammafuncder
==============================================
Purpose
----------------
Computes the derivative of the gamma function integral with respect to theta.

Format
----------------
.. function:: l = gammafuncder(theta)

    :param theta: Shape parameter.
    :type theta: scalar

    :return l: Computed derivative of the integral.
    :rtype l: scalar

Library
-------
bhatlib

Source
------
