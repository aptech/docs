sdrgumbel
==============================================
Purpose
----------------
Computes the standard deviation of the reversed Gumbel distribution.

Format
----------------
.. function:: sd = sdrgumbel(mu, beta)

    :param mu: Location parameter.
    :type mu: scalar

    :param beta: Scale parameter.
    :type beta: scalar

    :return sd: Standard deviation.
    :rtype sd: scalar

Library
-------
bhatlib
