
pqgwin
==============================================

Purpose
----------------

Sets the graphics viewer mode. NOTE: This function is for use only with the deprecated PQG graphics.

Format
----------------
.. function:: pqgwin one 
			  pqgwin many

Examples
----------------

::

    pqgwin many;
