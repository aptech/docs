
vartypef
==============================================

Purpose
----------------
Returns a vector of ones and zeros that
indicate whether variables in a data set are character or numeric.

Format
----------------
.. function:: vartypef(f)

    :param f: file handle of an open file.
    :type f: TODO

    :returns: y (*Nx1 vector of ones and zeros*), 1 if variable is
        numeric, 0 if character.
