
spTrTDense
==============================================

Purpose
----------------
Multiplies a sparse matrix transposed by a dense matrix.

Format
----------------
.. function:: spTrTDense(s, d)

    :param s: 
    :type s: NxM sparse matrix

    :param d: 
    :type d: NxL dense matrix

    :returns: y (*MxL dense matrix*), the result of  s'*d.
