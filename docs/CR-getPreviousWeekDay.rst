
getPreviousWeekDay
==============================================

Purpose
----------------

Returns the previous day that is not on a weekend.

Format
----------------
.. function:: getPreviousWeekDay(a)

    :param a: date in DT scalar format.
    :type a: scalar

    :returns: n (*scalar*), previous week day in DT scalar format.
