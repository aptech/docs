
lncdfnc
==============================================

Purpose
----------------

Computes natural log of complement of Normal cumulative distribution function.

Format
----------------
.. function:: y = lncdfnc(x)

    :param x: abscissae.
    :type x: NxK matrix

    :return y:
    
        .. math:: ln\ (1 - Pr(X < x))

    :type y: NxK matrix
