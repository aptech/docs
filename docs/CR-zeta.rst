
zeta
==============================================

Purpose
----------------
Computes the Riemann Zeta function.

Format
----------------
.. function:: f = zeta(z)

    :param z: data. *z* may be complex
    :type z: NxK matrix

    :return f: 

    :rtype f: NxK matrix

Remarks
-------

Euler MacLaurin series.

References
----------

#. Jon Breslaw, 2009
