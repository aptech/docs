
threadStat
==============================================

Purpose
----------------

Marks a single line of code to be executed as a thread.

.. _threadStat:
.. index:: threadStat

Format
----------------

::

    threadStat statement;

Examples
----------------

::

    threadStat m = n*p;

See `threadJoin` for an example of a fully-defined thread set.

.. seealso:: Functions `threadBegin`, `threadEnd`, `threadJoin`
