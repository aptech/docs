
strtrimr
==============================================

Purpose
----------------

Strips all whitespace characters from the right side of each element 
in a string array.

Format
----------------
.. function:: strtrimr(sa)

    :param sa: 
    :type sa: NxM string array

    :returns: y (*NxM string array*) .
