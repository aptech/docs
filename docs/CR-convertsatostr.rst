
convertsatostr
==============================================

Purpose
----------------

Converts a 1x1 string array to a string.

Format
----------------
.. function:: str = convertsatostr(sa)

    :param sa:
    :type sa: 1x1 string array

    :returns: **str** (*string*) - *sa* converted to a string.

.. seealso:: :func:`convertstrtosa`
