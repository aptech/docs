
eqSolveSet
==============================================

Purpose
----------------

Sets global input used by eqSolve to default values.

Format
----------------
.. function:: eqSolveset

    :returns: __eqs_TypicalX (*TODO*), Set to 0.

    :returns: __eqs_TypicalF (*TODO*), Set to 0.

    :returns: __eqs_IterInfo (*TODO*), Set to 0.

    :returns: __eqs_JacobianProc (*TODO*), Set to 0.

    :returns: __eqs_MaxIters (*TODO*), Set to 100.

    :returns: __eqs_StepTol (*TODO*), Set to __macheps2/3
