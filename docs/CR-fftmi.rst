
fftmi
==============================================

Purpose
----------------

Computes a multi-dimensional inverse FFT.

Format
----------------
.. function:: fftmi(x, dim)

    :param x: data.
    :type x: Mx1 vector

    :param dim: size of each dimension.
    :type dim: Kx1 vector

    :returns: y (*Lx1 vector*), inverse FFT of x.
