
endwind
==============================================

Purpose
----------------

Ends graphic panel manipulation; displays graphs with rerun. 

.. NOTE:: This function is for use with the deprecated PQG graphics.

Format
----------------
.. function:: endwind

Remarks
-------

This function uses rerun to display the most recently created ``.tkf`` file.
