
maxbytes
==============================================

Purpose
----------------

Returns maximum memory to be used.

Format
----------------
.. function:: maxbytes

    :param __maxbytes: maximum memory to be used.
    :type __maxbytes: scalar

    :returns: y (*scalar*), maximum memory to be used.

Examples
----------------

::

    y = maxbytes;
    print y;

::

    100000000.000
