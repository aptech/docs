
pvUnpack
==============================================

Purpose
----------------

Unpacks matrices stored in a structure of type PV.

Format
----------------
.. function:: pvUnpack(p1, m)

    :param p1: 
    :type p1: an instance of structure of type PV

    :param m: name of matrix, or integer, index of matrix.
    :type m: string

    :returns: x (*TODO*), MxN general matrix or MxM symmetric matrix or
        N-dimensional array.
