
time
==============================================

Purpose
----------------

Returns the current system time.

Format
----------------
.. function:: y = time()

    :return y: the current time in the
        order: hours, minutes, seconds, and hundredths of a second.

    :type y: 4x1 numeric vector

Examples
----------------

::

    print time;

::

       7.000000
      31.000000
      46.000000
      33.000000
