
pvList
==============================================

Purpose
----------------

Retrieves names of packed matrices in structure of type :class:`PV`.

Format
----------------
.. function:: n = pvList(p1)

    :param p1: an instance of structure of type :class:`PV`
    :type p1: struct

    :returns: n (*Kx1 string vector*), names of packed matrices.
