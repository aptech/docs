
begwind
==============================================

Purpose
----------------
Initializes global graphic panel variables. 

.. NOTE:: This function is for the deprecated PQG graphics.

Library
-------
pgraph

Format
----------------
.. function:: begwind

Remarks
-------

This procedure must be called before any other graphic panel functions are called.

------
