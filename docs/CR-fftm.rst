
fftm
==============================================

Purpose
----------------

Computes a multi-dimensional FFT.

Format
----------------
.. function:: fftm(x, dim)

    :param x: data.
    :type x: Mx1 vector

    :param dim: size of each dimension.
    :type dim: Kx1 vector

    :returns: y (*Lx1 vector*), FFT of x.
