
maxvec
==============================================

Purpose
----------------

Returns maximum vector length allowed.

Format
----------------
.. function:: maxvec

    :param __maxvec: maximum vector length allowed.
    :type __maxvec: scalar

    :returns: y (*scalar*), maximum vector length.

Examples
----------------

::

    y = maxvec;
    print y;

::

    20000.000
