
fn
==============================================

Purpose
----------------

Allows user to create one-line functions.

Format
----------------
.. function:: code_for_function

Remarks
-------

Functions created usig `fn` can be called in the same way as other procedures.


Examples
----------------

::

    fn area(r) = pi*r*r;
    a = area(4);

After the code above:

::

    a = 50.2625
