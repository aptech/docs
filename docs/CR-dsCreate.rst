
dsCreate
==============================================

Purpose
----------------

Creates an instance of a structure of type DS set to default values.

Format
----------------
.. function:: dsCreate()

    :returns: s (*TODO*), instance of structure of type DS.

Examples
----------------

::

    //Declare 'myData' as instance of 'DS' structure
    struct DS myData;
    
    //Apply default settings
    myData = dsCreate();
