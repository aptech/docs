
dsCreate
==============================================

Purpose
----------------

Creates an instance of a structure of type :code:`DS` set to default values.

Format
----------------
.. function:: s = dsCreate()

    :returns: **s** (*struct*) - instance of :class:`DS` struct.

Examples
----------------

::

    // Declare 'myData' as instance of 'DS' structure
    struct DS myData;

    // Apply default settings
    myData = dsCreate();
