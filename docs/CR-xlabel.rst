
xlabel
==============================================

Purpose
----------------
Sets a label for the X axis.

.. NOTE:: This function is for use with the deprecated PQG graphics, use plotSetXLabel for equivalent functionality.

Library
-------

pgraph

Format
----------------
.. function:: xlabel(str)

    :param str: the label for the X axis.
    :type str: string
