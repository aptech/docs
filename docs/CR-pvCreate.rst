
pvCreate
==============================================

Purpose
----------------

Returns an initialized instance of structure of type :class:`PV`.

Format
----------------
.. function:: p1 = pvCreate()

    :returns: p1 (*struct*) instance of :class:`PV` struct

Examples
----------------

::

    // Declare 'p1' as an instance of a 'PV' structure
    struct PV p1;
    
    // Fill in 'p1' with default values
    p1 = pvCreate();
