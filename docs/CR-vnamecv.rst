
vnamecv
==============================================

Purpose
----------------
Returns the names of the elements of a data buffer constructed with vput.

Format
----------------
.. function:: vnamecv(dbuf)

    :param dbuf: a data buffer containing various strings and matrices.
    :type dbuf: Nx1 vector

    :returns: cv (*Kx1 character vector*) containing the names of the elements of  dbuf.
