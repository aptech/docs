
utrisol
==============================================

Purpose
----------------

Computes the solution of Ux = b where  U is an upper
triangular matrix.

Format
----------------
.. function:: utrisol(b, U)

    :param b: 
    :type b: PxK matrix

    :param U: 
    :type U: PxP upper triangular matrix

    :returns: x (*PxK matrix*), solution of Ux = b.
