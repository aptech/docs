
numCombinations
==============================================

Purpose
----------------

Computes number of combinations of n things taken k at a time.

Format
----------------
.. function:: numCombinations(n,  k)

    :param n: scalar.
    :type n: TODO

    :param k: scalar.
    :type k: TODO

    :returns: y (*scalar*), number of combinations of  n things take  k at a time.

----------------

    y = numCombinations(25,5);
     
    print y;
