
sqpSolveSet
==============================================

Purpose
----------------

Resets global variables used by :func:`sqpSolve` to default values.

Format
----------------
.. function:: sqpSolveSet



Examples
--------

::

    // Reset sqpSolve global variables to defaults
    sqpSolveSet;

------
