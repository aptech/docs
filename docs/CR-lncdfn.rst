
lncdfn
==============================================

Purpose
----------------

Computes natural log of Normal cumulative distribution function.

Format
----------------
.. function:: y = lncdfn(x)

    :param x: abscissae.
    :type x: NxK matrix or N-dimensional array

    :return y: 
    
        .. math:: ln Pr(X < x)

    :type y: NxK matrix or N-dimensional array
