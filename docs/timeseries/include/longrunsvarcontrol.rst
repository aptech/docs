.. list-table::
   :widths: auto

   * - adv.include_const
     - Scalar, 1 to include a constant (intercept), 0 to exclude. Default = 1.

   * - adv.xreg
     - TxK matrix, exogenous regressors. Default = empty (no exogenous variables).

   * - adv.quiet
     - Scalar, set to 1 to suppress printed output. Default = 0.
