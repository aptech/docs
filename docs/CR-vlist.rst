
vlist
==============================================

Purpose
----------------
Lists the contents of a data buffer constructed with vput.

Format
----------------
.. function:: vlist(dbuf)

    :param dbuf: a data buffer containing various strings
        and matrices.
    :type dbuf: Nx1 vector



Remarks
-------

vlist lists the names of all the strings and matrices stored indbuf.
