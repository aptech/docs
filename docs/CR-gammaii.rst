
gammaii
==============================================

Purpose
----------------

Computes the inverse incomplete gamma function.

Format
----------------
.. function:: gammaii(a, p)

    :param a: exponents.
    :type a: MxN matrix

    :param p: ExE conformable with  a, incomplete gamma
        values.
    :type p: KxL matrix

    :returns: x (*max(M,K) by max(N,L) matrix*) , abscissae.
