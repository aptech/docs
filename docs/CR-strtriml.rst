
strtriml
==============================================

Purpose
----------------
Strips all whitespace characters from the left side of each element in a string array.

Format
----------------
.. function:: y = strtriml(sa)

    :param sa: data
    :type sa: NxM string array

    :returns: y (*NxM string array*)
