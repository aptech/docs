=========
permutate
=========

10.0.42permutate
================

Purpose
-------

.. container::
   :name: Purpose

   Lists all possible permutations without replacement for n number of
   items, chosen k times.

Library
-------

.. container:: gfunc
   :name: Library

   tsmt

Format
------

.. container::
   :name: Format

   y = permutate(n, k);

Input
-----

.. container::
   :name: Input

   = =================================
   n number of items.
   k number of times items are chosen.
   = =================================

Output
------

.. container::
   :name: Output

   = ===========================================
   y a matrix listing all possible permutations.
   = ===========================================

------
