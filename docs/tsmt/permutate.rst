=========
permutate
=========

10.0.42permutate
================

Purpose
-------
Lists all possible permutations without replacement for n number of
   items, chosen k times.

Library
-------
tsmt

Format
------
y = permutate(n, k);

Input
-----
= =================================
   n number of items.
   k number of times items are chosen.
   = =================================

Output
------
= ===========================================
   y a matrix listing all possible permutations.
   = ===========================================

------
permutate.src
