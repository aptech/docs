===================
lsdvmtControlCreate
===================

10.0.35lsdvmtControlCreate
==========================

Purpose
-------
Sets the members of an instance of an lsdvmtControl structure to
   default values.

Library
-------
tsmt

Format
------
lsc = lsdvmtControlCreate( );

Input
-----
None

Output
------
+-----+---------------------------------------------------------------+
   | lsc | An instance of an lsdvmtControl structure with its members    |
   |     | set to default values.                                        |
   +-----+---------------------------------------------------------------+

Source
------
lsdvmt.src
