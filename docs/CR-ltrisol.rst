
ltrisol
==============================================

Purpose
----------------

Computes the solution of Lx = b where L is a lower triangular matrix.

Format
----------------
.. function:: ltrisol(b, L)

    :param b: 
    :type b: PxK matrix

    :param L: 
    :type L: PxP lower triangular matrix

    :returns: x (*PxK matrix*), soluion of Lx = b.
