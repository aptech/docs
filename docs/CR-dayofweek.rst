
dayofweek
==============================================

Purpose
----------------

Returns day of week.

Format
----------------
.. function:: dayofweek(a)

    :param a: dates in DT format.
    :type a: Nx1 vector

    :returns: d (*Nx1 vector*), integers indicating day of week of each date:

    .. csv-table::
        :widths: auto

        "1", "Sunday"
        "2", "Monday"
        "3", "Tuesday"
        "4", "Wednesday"
        "5", "Thursday"
        "6", "Friday"
        "7", "Saturday"
