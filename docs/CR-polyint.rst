
polyint
==============================================

Purpose
----------------

Calculates an Nth order polynomial interpolation.

Format
----------------
.. function:: polyint(xa, ya, x)

    :param xa: x values.
    :type xa: Nx1 vector

    :param ya: y values.
    :type ya: Nx1 vector

    :param x: x value to solve for.
    :type x: scalar

    :returns: y (*TODO*), result of interpolation or extrapolation.
