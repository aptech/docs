
qrsol
==============================================

Purpose
----------------

Computes the solution of Rx = b where R is an
 upper triangular matrix.

Format
----------------
.. function:: qrsol(b, R)

    :param b: 
    :type b: PxL matrix

    :param R: 
    :type R: PxP upper triangular matrix

    :returns: x (*TODO*), PxL matrix.
