
getNextWeekDay
==============================================

Purpose
----------------

Returns the next day that is not on a weekend.

Format
----------------
.. function:: getNextWeekDay(a)

    :param a: date in DT scalar format.
    :type a: scalar

    :returns: n (*scalar*), next week day in DT scalar format.
