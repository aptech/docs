
iscplxf
==============================================

Purpose
----------------

Returns whether a data set is complex or real.

Format
----------------
.. function:: iscplxf(fh)

    :param fh: file handle of an open file.
    :type fh: scalar

    :returns: y (*scalar*), 1 if the data set is complex, 0 if it is real.

.. seealso:: Functions :func:`hasimag`, :func:`iscplx`
