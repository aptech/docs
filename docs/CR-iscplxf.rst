
iscplxf
==============================================

Purpose
----------------

Returns whether a dataset is complex or real.

Format
----------------
.. function:: y = iscplxf(fh)

    :param fh: file handle of an open file.
    :type fh: scalar

    :return fh_iscplx: 1 if the data set is complex, 0 if it is real.

    :type fh_iscplx: scalar

.. seealso:: Functions :func:`hasimag`, :func:`iscplx`
