
outwidth
==============================================

Purpose
----------------

Specifies the width of the auxiliary output.

Format
----------------
.. function:: outwidth n

    :param n: width of auxilary output.
    :type n: scalar

Examples
----------------

::

    outwidth 132;

This statement will change the auxiliary output width to 132 columns.
