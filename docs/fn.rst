
fn
==============================================

Purpose
----------------

Allows user to create one-line functions.

.. _fn:
.. index:: fn

Format
----------------

::

    fn ret = code_for_function;


Examples
----------------

::

    fn area(r) = pi*r*r;
    a = area(4);

After the code above:

::

    a = 50.2625

Remarks
-------

Functions created using `fn` can be called in the same way as other procedures.
